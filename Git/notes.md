### Levels

- **System:** All users
- **Global:** All repositories of the current user
- **Local:** The current repository

```bash
git config --global user.name "SongsongTang"
git config --global user.email "tss1375716572@163.com"
git config --global core.editor "neovim"
git config --global -e  # open global settings using default editor
```
